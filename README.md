# Operating Systems Coursework

This repository collects my source code and submitted PDF work from an operating systems course.

## Contents

- `Assignments/HW1`: C programs and the HW1 PDF
- `Assignments/HW2`: CPU scheduling implementations in C and Java, supporting build files, and the HW2 PDF
- `Assignments/HW3`: Producer-consumer and page-offset C programs
- `Assignments/HW4`: Virtual-memory-manager C program
- `Assignments/Chapter 12 Questions.pdf` and `Assignments/Chapter 13 Questions.pdf`: written assignment PDFs

The directory structure and original coursework filenames have been preserved. Compiled binaries, object files, archives, screenshots, textbook files, and lecture materials are intentionally excluded.
