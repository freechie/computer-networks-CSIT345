# Operating Systems Coursework

This repository collects my source code and submitted PDF work from an operating systems course.

## Contents

- `HW1`: C programs and the HW1 PDF
- `HW2`: CPU scheduling implementations in C and Java, supporting build files, and the HW2 PDF
- `HW3`: Producer-consumer and page-offset C programs
- `HW4`: Virtual-memory-manager C program

Original coursework filenames have been preserved. Compiled binaries, object files, archives, screenshots, textbook files, and lecture materials are intentionally excluded.
